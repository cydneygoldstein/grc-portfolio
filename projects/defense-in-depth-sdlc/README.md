# Defense-in-Depth Plan for Software Development Lifecycle

This project is a sanitized secure SDLC defense-in-depth case study developed for a fictional enterprise, Humongous Holdings. It presents a governance-focused roadmap for strengthening software development lifecycle controls, integrating security throughout every development phase, improving third-party oversight, and formalizing documentation, testing, and configuration management practices.

The plan was developed using NIST 800-53, NIST CSF, NIST RMF, FIPS 199, and FIPS 200 as primary reference points. Recommendations were prioritized by impact level and structured to address both technical and administrative weaknesses across the enterprise software development lifecycle.
